# 日割りダイエット

毎日の目標体重が一目でわかる！日割りダイエットで無理なく理想の体重へ。

## 概要

目標体重までの減量を日割りで管理する、シンプルで続けやすいダイエットアプリです。開始日から終了日までの期間を設定すると、毎日の目標体重を自動計算。カレンダーで進捗を一目で確認できます。

## 主な機能

- **日割り目標の自動計算**: 現在の体重、目標体重、開始日、終了日を設定するだけで、1日あたりの減量目標と毎日の目標体重を自動計算
- **カレンダーで進捗管理**: 色分けで進捗が一目でわかる（青：達成、赤：未達成、黄：終了日）
- **シンプルな体重記録**: 日付をタップするだけで体重を入力
- **スワイプで月移動**: カレンダーを左右にスワイプして簡単に月移動
- **柔軟な期間設定**: 開始日と終了日を自由に設定可能

## 技術スタック

- HTML/CSS/JavaScript
- Capacitor (iOS)
- LocalStorage (データ保存)

## Privacy Policy

### プライバシーポリシー

最終更新日: 2026年4月5日

#### データの収集について

「日割りダイエット」（以下「本アプリ」）は、ユーザーのプライバシーを尊重します。

**本アプリ自体はユーザーのデータを収集しません。** すべての体重記録データは端末内にのみ保存されます。

#### 広告について

本アプリはGoogle AdMobを使用して広告を表示しています。AdMobは広告配信のために以下の情報を収集する場合があります：

- デバイスの識別情報
- 広告の表示・クリックに関するデータ
- IPアドレス

詳細はGoogleのプライバシーポリシー（https://policies.google.com/privacy）をご確認ください。

#### データの保存について

- すべてのデータ（体重記録、目標設定など）は、お使いの端末内にのみ保存されます
- データはインターネット経由で送信されることはありません
- 開発者がユーザーのデータにアクセスすることはありません

#### 第三者へのデータ共有

本アプリは、第三者とデータを共有することは一切ありません。

#### データの削除

アプリをアンインストールすることで、すべてのデータが端末から削除されます。
